# Amend

Amend creates and maintains local, Git-backed knowledge wikis from source
documents.

## Language

**Wiki**:
A local knowledge base created, updated, and viewed in Amend.
_Avoid_: Workspace, knowledge base

**Wiki home**:
The user-selected directory containing Amend's sibling wiki repositories.
_Avoid_: Workspace home, library

**Wiki engine**:
The ingestion and indexing capability that writes and searches a wiki.
_Avoid_: Wiki workspace
